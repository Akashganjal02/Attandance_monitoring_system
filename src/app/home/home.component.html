Attendance Monitoring System

Attendance Monitoring System is a web application developed for daily student attendance in schools, colleges and institutes. It facilitates to access the attendance information of a particular student in a particular class and shift. The information is sorted by the operators, which will be provided by the staff for a particular class. This system will also help in evaluating attendance criteria of a student.
